# Amazon Product Analysis

## Objective
To analyze Amazon product data and derive insights releated to the product, discount, ratings and customer preferences.

## Dataset
Amazon product dataset sourced from Kaggle

## Tools Used
- Python(Pandas, Matplotlib)
- MySQL(SQL queries for analysis)
- Excel(Pivot Tables and charts)

## Key Analysis Performed
- Data Cleaning and preprocessing
- Category Wise rating and discount Analysis
- Product Popularity analysis using rating counts
- Visualization of trends using Python and Excel

## Business Insights
- Certain categories consistently receive higher ratings.
- High discounts do not always correlate with higher ratings.
- Popular products are often moderately priced rather than premium.
- Customer preference is influenced more by product quality than discount alone.

## Conclusion
The project demonstrates how data analysis and SQL can be used together to support business decision-making.
